# Typing Party App

まず、`pip install -r requirements.txt` をする。

その次に`python main.py`を実行する。

# Typing Party Appの遊び方

10個問題がランダムに出題されるので間違いなく答えてください。
10問答えたら終了です。終了までの時間と正解数を表示してくれます。

再スタートを押すともう一回最初から始めれます。

タイピングを鍛えたい方にはおススメのGUIアプリとなっています。
